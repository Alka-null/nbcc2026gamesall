# Audio Assets

## Required Files:

1. **background_music.mp3** - Upbeat instrumental background music (loops)
2. **click.mp3** - Puzzle piece pick/drop sound
3. **correct.mp3** - Piece snaps into correct position
4. **success.mp3** - Puzzle completion celebration
5. **game_start.mp3** - Game start jingle

## Specifications:
- Format: MP3
- Background music: 2-3 min loop, kid-friendly instrumental
- Sound effects: 50-300ms short clips
- Volume controlled by app (30% music, 50% effects)
