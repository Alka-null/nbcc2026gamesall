# Image Assets

## Required Files:

1. **heineken_bg.jpg**
   - Heineken bottle background image
   - Used as styled background throughout the game
   - Fallback: Green gradient if missing

2. **evergreen_preview.png**
   - Evergreen 2030 logo (circular design)
   - Shows as animated preview before puzzle starts
   - Displays with glowing effects and twinkling stars
   - Recommended size: 300x300 or larger
   - Fallback: Gradient with text/icon if missing

## Instructions:
1. Copy the Heineken background as `heineken_bg.jpg`
2. Save the Evergreen 2030 logo as `evergreen_preview.png`
