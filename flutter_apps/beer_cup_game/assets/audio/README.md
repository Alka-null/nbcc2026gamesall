# Audio Assets

## Required Files:

1. **background_music.mp3** - Upbeat instrumental background music (loops)
2. **click.mp3** - Button click sound effect
3. **correct.mp3** - Correct answer chime
4. **wrong.mp3** - Wrong answer buzz
5. **success.mp3** - Game completion celebration
6. **game_start.mp3** - Game start jingle

## Specifications:
- Format: MP3
- Background music: 2-3 min loop, 120-140 BPM
- Sound effects: 50-300ms short clips
- Volume will be controlled by the app (30% music, 50% effects)
